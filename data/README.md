# Data

## `raw/`
Fastq files from particulate matter (PM2.5) samples collected in Medellín over an 18-month period.

These files are not included in this repository due to their large size.
To request access to the raw data, please contact: juan.vargasg1@udea.edu.co

## `metadata/`
Information associated with each sample.

- `Metadata_PM.csv`: Table containing the list of samples, date collection, period, number of sample and PM2.5 concentration (µg/m³).
